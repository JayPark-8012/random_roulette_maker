# PRD - 랜덤 룰렛 메이커 (Random Roulette Maker)

## 1. 제품 개요

### 목표
누구나 5초 안에 나만의 룰렛을 만들고, 바로 돌릴 수 있는 모바일 앱.
복잡한 설정 없이 항목만 입력하면 즉시 사용 가능한 '즉석 결정 도구'.

### 타겟 사용자
- **1차 타겟**: 점심 메뉴, 술자리 게임, 팀 역할 배분 등 일상적 결정이 필요한 20~30대
- **2차 타겟**: 교육 현장에서 발표자/팀 선택이 필요한 교사·강사
- **3차 타겟**: 유튜브/스트리밍 방송에서 시청자 참여 이벤트를 진행하는 크리에이터

### 핵심 가치 제안
- 빠름: 앱 실행 후 3탭 이내에 스핀 가능
- 유연함: 항목 수 제한 없이 자유롭게 구성
- 재미: 스핀 애니메이션 + 사운드 + 진동으로 몰입감 제공

## 2. 핵심 기능 (MVP)

### 2.1 룰렛 관리
| 기능 | 설명 |
|------|------|
| 룰렛 생성 | 이름 + 항목(최소 2개) 입력 후 저장 |
| 룰렛 편집 | 이름/항목 수정, 색상 변경 |
| 룰렛 삭제 | 스와이프 또는 편집 화면에서 삭제 |
| 템플릿 사용 | 10개 사전 제작 템플릿에서 복사 후 바로 사용 |

### 2.2 스핀(Play)
| 기능 | 설명 |
|------|------|
| 룰렛 스핀 | 탭 한 번으로 회전 시작, 물리 감속 |
| 결과 표시 | 중앙 포인터가 가리키는 항목 강조 + 결과 모달 |
| 재스핀 | 결과 모달에서 바로 다시 돌리기 |
| 히스토리 | 룰렛별 최근 20개 결과 저장 |

### 2.3 사용자 설정
| 기능 | 설명 |
|------|------|
| 사운드 ON/OFF | 스핀 중 효과음 토글 |
| 진동 ON/OFF | 결과 발표 시 진동 토글 |

### 2.4 공유 (MVP 최소 범위)
| 기능 | 설명 |
|------|------|
| 결과 텍스트 복사 | "오늘의 점심: 짜장면" 형식으로 클립보드 복사 |
| 스크린샷 안내 | "스크린샷으로 공유하세요" 토스트 메시지 |

## 3. 비기능 요구사항

### 3.1 오프라인 동작
- 모든 기능이 인터넷 없이 완전히 동작해야 함
- 로컬 저장소(SharedPreferences 또는 파일 시스템)만 사용
- 초기 앱 설치 후 네트워크 요청 없음

### 3.2 성능
- 앱 콜드 스타트: 2초 이내 (Splash → Home)
- 스핀 애니메이션: 60fps 유지
- 룰렛 목록 로드: 100ms 이내 (최대 100개 항목 기준)

### 3.3 접근성
- 최소 텍스트 크기: 14sp
- 색상 대비: WCAG AA 기준 (4.5:1 이상)
- 시멘틱 레이블: 주요 버튼 및 아이콘에 `Semantics` 위젯 적용
- 다크 모드: 시스템 설정 연동 (MVP 후 단계)

### 3.4 기기 지원
- Android: API 24 이상 (Android 7.0+)
- iOS: iOS 13 이상
- 화면 크기: 360dp~430dp 너비 대응

## 4. 무료 / 프리미엄 정책

### 4.1 무료 티어 (MVP 전용)
| 항목 | 제한 |
|------|------|
| 룰렛 개수 | **최대 3개** |
| 룰렛당 항목 수 | 무제한 |
| 히스토리 | 룰렛별 최근 20개 |
| 광고 | 미정 (MVP에서는 광고 없음) |
| 템플릿 사용 | 전체 10개 무료 |

### 4.2 프리미엄 티어 (Post-MVP, Phase 3)
| 항목 | 내용 |
|------|------|
| 룰렛 개수 | 무제한 |
| 클라우드 백업 | Google Drive 연동 |
| 고급 테마 | 커스텀 색상, 폰트 |
| 히스토리 | 무제한 |

### 4.3 제한 도달 시 UX
- 4번째 룰렛 생성 시도 시: "무료 플랜은 최대 3개까지 가능합니다" BottomSheet 표시
- BottomSheet에 "프리미엄 알아보기" CTA 버튼 (MVP에서는 비활성화 또는 준비 중 안내)
- 기존 3개 룰렛 삭제 후 재생성은 허용
