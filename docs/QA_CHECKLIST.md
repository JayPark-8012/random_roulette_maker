# QA_CHECKLIST - 랜덤 룰렛 메이커

## 테스트 환경 기준
- 기기: 실기기 1대 이상 (Android, iOS 각 1대)
- 화면 크기: 소형(5") + 대형(6.5") 각 1회
- OS: Android 8.0+, iOS 14+
- 다크 모드: 별도 표기 없으면 라이트 모드 기준

## 1. 생성 (Create) 테스트

| # | 테스트 케이스 | 예상 결과 | 통과 |
|---|-------------|---------|-----|
